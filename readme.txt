== Description ==

**Notification bar.**

- Notification bar to easily add any webpage o video: youtube, vimeo...to your site. Demo: http://socialtvs.net 
- Barra de notificaciones para añadir fácilmente cualquier página web o vídeos youtube, vimeo... a tu sitio. Demo: http://socialtvs.net

[youtube https://www.youtube.com/watch?v=7ChbS9bzh0A]  

== Installation ==

- Install from WP Dashboard>>Plugins>>Add new plugin>>Search for 'SocialTVs' plugin. Activate the plug-in & configure webpages.
Any webpage that can be embedded will fit. 
* Example with youtube, copy/paste the iframe url: http://www.youtube.com/embed/kIqO1DPOkR0?autoplay=1 
* Automatically start playing your videos with ?autoplay=1 on youtube.
- Instala desde WP Dashboard>>Plugins>>Añadir nuevo>>Busca el plugin 'SocialTVs'. Actívalo. Configúralo: 
Por ejemplo, con youtube, copia/pega la url iframe de Compartir/Insertar: http://www.youtube.com/embed/kIqO1DPOkR0?autoplay=1
